# text-editor

I created a PWA Text Editor that runs in a browser, and can be installed by clicking an 'Install!' button.

## Table of Contents
  
* [Description](#Description)
* [Technologies](#Technologies)
* [Screenshots](#Screenshots)

## Description

Upon visiting the website on Heroku, users can
* Write code using the text editor, both online and offline
* Install the app and use it on a local system upon install
